## Deployment notes

Set the Airwallex and email variables in Vercel before testing live payments:

- `AIRWALLEX_CLIENT_ID`
- `AIRWALLEX_API_KEY`
- `AIRWALLEX_ENV`
- `SITE_URL`
- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_USER`
- `SMTP_PASS`
- `SMTP_SECURE`
- `EMAIL_FROM`
- `WELLBEING_NOTIFY_EMAIL`
